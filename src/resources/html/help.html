General

File menu

  • Export - 3D LUT (.cube) generation from the current pipeline

Global shortcuts

  • Esc - exit application

Image viewer shortcuts

  • Double click or Backspace - reset view
  • Click - control slider before/after position
  • ⌘ + Click - move image
  • Alt + click - center view on mouse position

Dev

This view is composed of :

  • Image view : pipeline render
  • Pipeline : this is were you put operators that will define the look pipeline
    Accept drag and drop from the operator list or directly from files
  • Operator list : available operators are shown here, you can drag and drop to the pipeline
  • Operator detail : when an operator is selected, a list of controls will appear, composed of both generic adjustement (global opacity, color, contrast, ...) and specifics ones related to the operator
  • Scope view : waveform, parade scope, values can go below or above 0..1
  • Look selection : Work in progress...

Available operators :

  • CTL Transform - Work in progress...
  • OCIO Colorspace - Require a .ocio configuration file, choose input / destination colorspace with optional look
  • OCIO File Transform - Require a LUT (all formats supported by OCIO)
  • OCIO Matrix - Work in progress...

Pipeline shortcuts

  • D - enable / disable selected operator
  • Backspace - delete selected operator

Scope shortcuts

  • +, - - increase / decrease opacity
  • All image viewer shortcuts available (zoom, ...)

Look

This view is composed of :

  • Look view : list of look in the current selected folder, can open multiple tabs
  • Look browser : show the content of the look folder (see. Setting)
  • Look search bar : filter through existing looks
  • Look detail : when a look is selected from the central or selection view, show image and curve preview
  • Look selection : drag and drop look from the central view here to make a selection
    Can clear / save / load selection to plain text files
    Selected looks will appear in comparison mode on the detail view
    Can deselect by clicking on empty space

Tab shortcuts

  • Spacebar - switch full detail view on / off
  • ↑, ↓ - navigate in the look view list
  • Shift + ↑, ↓ - navigate in the look selection list

Setting

General

  • Default image - this image will be used for the Dev and Look tabs
  • Look Base Folder - folder where the looks are stored
  • Look Tonemap LUT - optional tonemapping LUT applied after individual looks in the Look tab
